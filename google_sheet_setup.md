# วิธีเชื่อมเครื่องมือกับ Google Sheet

## 1. สร้าง Google Sheet
สร้างชีตใหม่ ตั้งชื่อ เช่น "บันทึกตรวจตลาดสด" — แถวแรกใส่หัวคอลัมน์:

```
วันที่ | ผู้ตรวจ | หมวด | สถานที่/โซน | รายการ | ผลตรวจ | หมายเหตุ | รูปภาพ1 | รูปภาพ2 | เวลาบันทึก
```

## 2. สร้างชีตรายชื่อ รปภ. (สำหรับ dropdown)
สร้างชีตใหม่อีกแผ่นในไฟล์เดียวกัน ตั้งชื่อแผ่นว่า **`setting`** (ตัวพิมพ์เล็กทั้งหมด)
- แถว A1: ใส่หัวข้อ เช่น "ชื่อ รปภ." (ไม่บังคับ)
- แถว A2 เป็นต้นไป: ใส่ชื่อ-นามสกุล รปภ. แต่ละคน แถวละ 1 ชื่อ

เครื่องมือจะดึงรายชื่อจากชีตนี้มาแสดงเป็น dropdown ให้เลือกตอนเปิดใช้งานอัตโนมัติ (ถ้าจะเพิ่ม/ลบชื่อ แค่แก้ในชีตนี้ ไม่ต้องแก้โค้ด)

## 3. เปิด Apps Script
เมนู Extensions > Apps Script แล้วลบโค้ดเดิมในไฟล์ Code.gs ทั้งหมด แล้วคัดลอกโค้ดจากไฟล์แนบ **`AppsScript_doPost.gs`** ไปวางแทน (ไฟล์นี้แยกไว้ต่างหากเพื่อให้ส่งต่อให้ทีมงานติดตั้งได้ง่าย)

## 4. Deploy เป็น Web App
1. กด Deploy > New deployment
2. เลือกประเภท "Web app"
3. Execute as: **Me**
4. Who has access: **Anyone**
5. กด Deploy แล้วอนุญาตสิทธิ์ที่ขอ (อนุมัติ Drive + Sheet)
6. คัดลอกลิงก์ที่ลงท้ายด้วย `/exec`

## 5. ใส่ลิงก์ในไฟล์เครื่องมือ
เปิดไฟล์ `market_inspection_tool.html` ด้วยโปรแกรมแก้ไขข้อความ ค้นหาคำว่า `SHEET_URL` (อยู่ต้น `<script>`) แล้วแทนที่ลิงก์ `/exec` ด้วยลิงก์ของคุณ:

```javascript
const SHEET_URL = "https://script.google.com/macros/s/XXXXXXXX/exec";
```

บันทึกไฟล์ แล้วเปิดใช้งานได้ตามปกติ — ไม่มีหน้าตั้งค่าให้ผู้ใช้กรอกลิงก์เอง (ลิงก์ฝังไว้ในตัวเครื่องมือแล้ว)

กดปุ่ม **"บันทึกข้อมูล"** ในแต่ละแท็บ เพื่อส่งข้อมูลที่ตรวจแล้วของวันนั้นเข้า Google Sheet (รวมรูปภาพที่อัปโหลดไป Google Drive อัตโนมัติ และชื่อผู้ตรวจที่เลือกไว้ด้านบน)

## หมายเหตุ
- ต้อง deploy ใหม่ทุกครั้งที่แก้โค้ด Apps Script (เลือก "New deployment" หรือ "Manage deployments" > แก้ไขเวอร์ชัน)
- รูปภาพจะถูกบีบอัดก่อนส่ง เพื่อไม่ให้ข้อมูลโตเกินไป
- ถ้าส่งไม่สำเร็จ ให้ตรวจสอบว่าลิงก์ลงท้ายด้วย `/exec` และตั้งค่า "Anyone" ในขั้นตอน Deploy
- ถ้าจะใช้กับหลายอุปกรณ์/รปภ.หลายคน ให้แก้ `SHEET_URL` ในไฟล์ก่อนแจกจ่ายให้แต่ละเครื่อง (ลิงก์เดียวกันใช้ร่วมกันได้)
- ถ้าเคย deploy เวอร์ชันเก่าไว้แล้ว และนำโค้ด `AppsScript_doPost.gs` เวอร์ชันใหม่ (มี `doGet` สำหรับโหลดรายชื่อ รปภ.) ไปวางทับ ต้อง deploy เวอร์ชันใหม่ด้วย (Manage deployments > แก้ไข > New version > Deploy) ไม่เช่นนั้น dropdown จะไม่มีรายชื่อ
